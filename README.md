# AutoState
> АвтоСтатус для ВКонтакте. 

# Установка
1. Установим зависимости:

   Необходим Python3:
   
   
      ```pip3 install vk_api```
 

2.Установим Git:
  Для Linux:

      sudo apt-get install git
      
   Для Termux:

      pkg install git
      
3.Запускаем скрипт:

   Для Linux:
   
      sudo python3 main.py
      
      
   Для Termux:
   
      python3 main.py
